## What is rShell?

rShell is a minimal yet functional shell made in rust.

## Install

First of all install rust using [rustup](rustup.rs)

Second, clone the github repo:

```
git clone https://www.github.com/micziz/rshell.git
```


Third, cd and run it:

```
cd rshell
cargo run
``
